

platy_range_fade fades hostile nameplates when out of range hostile nameplates when enemies are outside your spell range. Designed to work alongside the Platynator nameplate addon, but also works on default blizzard nameplates.

---

Slash commands

| Command | Description |
|---|---|
| `/prf` | Open the settings window |
| `/prfade` | Open the settings window (alias) |
| `/prf debug` | Open the debug log window (scrollable, copy-pasteable) |
| `/prf reset` | Reset all settings to defaults (restores all default spell IDs, alpha 0.3, enabled) |

---

Settings

# Out-of-range alpha (set from 0 to 1)

# Tick rate (set from 0.05 to 1)
- this is the rate at which the addon scans for range. Decrease this if you encounter FPS spikes driven by PRF. From my testing, 0.15 worked well in raid with a ton of mobs, but I'm sure I'll get complaints anyway.

# Spell ID per spec
A scrollable list of all 40 retail specs, grouped by class. Each row shows the spec name, a spell ID input box, and the resolved spell name.
